# Nexus Calc — Futuristic Scientific Calculator

**Nexus Calc** is a neon-styled scientific calculator built with HTML, CSS and JavaScript.
It supports trig (sin/cos/tan with degrees/radians), log, ln, factorial, sqrt, powers, π, and e.

## How to run
1. Clone or download the repo.
2. Open `index.html` in your browser.

## Files
- `index.html` — UI
- `style.css`  — styles
- `script.js`  — calculator logic
